HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Linking CSS files to HTML can be done via a link tag and href alongside it's relationship "rel".
  • Scripts should be linked in HTML after the footer, but still in the body of your code.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on and if any changes have been made since last commit.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git checkout branch-name: switches to desired branch.
  • git add -A: stages all changes made in current working branch, ready for commit.
  • git commit -m "message": commits changes that were added to staging along with a message.
  • git pull origin main: pulls changes from origin remote, "main" branch and merges locally to checked-out branch.
  • git pull origin/main: pulls changes from locally cached "main" branch and merges locally to checked-out branch.
  • git push origin main: pushes committed changes to remote, "main" branch, beginning the merge process on remote repo.
  • git branch: lists all available branches and highlights the current working branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
